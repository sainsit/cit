About

Central Intelligence of Technology (CIT) satu satunya ekstrakurikuler yang mengunsung tema IT. CIT terbentuk tanggal 18 September 2019. Diresmikannya CIT pada tahun 2019 dengan menunggu kebijakan kesiswaan untuk surat keterangan sebagai ekstrakurikuler. Rencana awal CIT sendiri sudah ada sejak 2015, tetapi pada saat itu tidak terpikirkan sebagai ekstrakurikuler.

Apa yang kita pelajari?

Networking

Networking mempelajari bagian bagian penting dalam jaringan komputer khususnya bagian server, mulai dari DNS server untuk mengubah IP Address menjadi nama domain, DHCP server untuk menyebarkan alamat IP secara otomatis, server untuk memberikan layanan komunikasi data antar jaringan, dan Web Server untuk komunikasi penghubung dengan situs web dan memproses HTTP request yang dikirimkan oleh browser.

Tool

  • Terminal

Programmimg

Programming mempelajari pemrograman website, mulai dari website statis maupun website dinamis. Bagian depan website yang biasa disebut dengan front-end/client-side ditulis menggunakan Hypertext Markup Language(HTML), dan di design menggunakan Cascading Style Sheet(CSS), kita juga menggunakan JavaScript(JS) agar website lebih interaktif. Bagian belakang website yang biasa disebut back-end/server-side ditulis menggunakan bahasa pemrograman PHP:Hypertext Processor dan menggunakan database MySQL sebagai sarana penyimpanan data.

Language

  • HTML5

  • CSS3

  • JS

  • PHP

  • MySQL
